# 📊 Madhav Ecommerce Sales Dashboard

## Overview

This project is an interactive Power BI dashboard developed to analyze and monitor e-commerce sales performance. The dashboard provides valuable insights into sales, profit, customer behavior, payment methods, and product categories, helping businesses make data-driven decisions.

## Features

* Interactive dashboard with dynamic filters and slicers
* Quarter-wise and State-wise sales analysis
* KPI Cards for:

  * Total Sales Amount
  * Total Quantity Sold
  * Total Profit
  * Average Sales
* Sales analysis by State and Customer
* Category-wise quantity analysis
* Payment mode analysis
* Monthly profit trend analysis
* Sub-category profit performance tracking

## Tools & Technologies

* Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Data Modeling
* Data Visualization

## Key Insights

* Clothing category contributed the highest quantity sold.
* COD was the most preferred payment mode.
* Maharashtra generated the highest sales revenue.
* Printers delivered the highest profit among all sub-categories.
* Monthly profit analysis helped identify profitable and loss-making periods.
